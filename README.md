## 背景设计

### 地理

此方修仙大陆名为 玄黄大陆 ，其中包含5片大区：

- 中央：中土神州
  - 风格：礼乐文明，王道教化。这里是大陆灵气最均衡、文明最鼎盛之地。
  - 核心势力：大胤王朝与儒家书院共治。皇朝以“人道气运”统御万民，镇压山河；儒家书院则秉持“修身治国平天下”，是秩序与礼法的维护者。
  - 地貌与资源：千里沃野，雄城巨邑，龙脉汇聚。盛产能稳定心境、辅助“治国类”功法修炼的 “浩然石” 与 “灵谷”。
  - 核心矛盾：秩序与变数。皇朝与书院力图维持稳定，视一切魔道与过激的修真者为“变数”与“祸源”。这里是“正统”的代名词，也是风暴酝酿的中心。
- 东方：东极青霖
  - 风格：木系长生，丹符器盛。浩瀚林海覆盖，生机盎然，多灵脉福地。
  - 核心势力：以 “青帝谷” 为首的丹道宗门，和以 “天工坊” 为代表的炼器世家。他们与自然共生，追求长生与技艺极致。
  - 地貌与资源：万年灵木森林、翡翠湖泊、悬浮灵山。盛产各类奇花异草（炼丹）、灵木宝玉（炼器）以及 “生生不息”的木系先天灵气。
  - 核心矛盾：开发与守护。宗门家族间为争夺珍稀资源明争暗斗，同时需警惕过度开发触怒森林深处的古老木灵或妖王。
- 西方：西漠焚天
  - 风格：金火炼狱，苦行剑修。无垠沙漠与火山群交织，环境极端，灵气中充满锐利的金气与暴烈的火气。
  - 核心势力：信奉“以杀证道”、“千锤百炼”的 “绝剑宗” 等剑修门派，以及崇拜火焰与力量的 “拜火教” 。
  - 地貌与资源：赤金沙漠、熔岩裂谷、废弃上古矿坑。盛产 “庚金之精”（炼飞剑）、“地心火莲”（炼体）及各种高纯度金属矿石。
  - 核心矛盾：生存与毁灭。资源点稀少而珍贵，争夺无比残酷。这里盛产最锋利的剑，也孕育最暴烈的火和最坚韧的修士。
- 南方：南离炎域
  - 风格：巫蛊奇诡，血脉图腾。湿热雨林、毒瘴沼泽与活火山并存，充满原始、神秘的生命力。
  - 核心势力：传承上古巫道的 “百蛊教” 、供奉凤凰或凶兽的 图腾部落，以及掌控地火熔池的 “炎魔族”（非人智慧种族）。
  - 地貌与资源：盘虬雨林、七彩毒沼、永不熄灭的圣火祭坛。盛产 蛊虫、异毒、血脉果实（激发血脉力量）及 地火核心。
  - 核心矛盾：文明与野蛮，融合与排斥。人族巫道、部落文明与非人种族在此地碰撞、共存、争斗，信仰与血脉的力量远超简单的正魔之分。
- 北方：北冥冻土
  - 风格：冰魂死寂，魂修道场。万里冰原，永夜极光，是生与死的界限最为模糊之地。
  - 核心势力：修炼神魂、操控冰雪的 “广寒宫”，研究生死奥秘的 “黄泉道”（非纯粹魔道，更接近中立研究狂），以及沉睡在冰川下的 上古妖族。
  - 地貌与资源：万载玄冰平原、极光魂海、上古战场遗迹。盛产 “万年玄冰”（炼器、镇魂）、“冰魄”（修魂）及 “战魂英灵”（可收服或炼化）。
  - 核心矛盾：寂灭与复苏。这里是放逐之地、遗忘之地，也是寻求突破生死禁忌者的终极试验场。最大的危险往往来自寂静本身和那些不该被唤醒的存在。

### 纪年

纪年逻辑：以最后一位被公认的、成功飞升仙界的修仙者命名，并以飞升之年为纪元之始。
今年是芜苍五百一十二年，距离最后一位飞升者芜苍道人飞升已经过了五百一十二年，这五百余年里灵气愈加稀薄，再无飞升者，进入了飞升断绝状态。
这是最正式、最官方的纪年方式，见于史书、大宗门典籍与正式盟约。
